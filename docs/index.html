Before you begin

This is an iOS class. To complete the coding exercises, you will need a Xamarin.iOS development environment with either a configured iOS Simulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

The goal of this clear is to teach you how to create reactive IU using Auto Layout and Constraints. We will apply constraints to views both in the iOS Designer and in code. And we'll use Size Classes to adjust our UI for different screen sizes. By the end of this course, you'll be able to create adaptive user interfaces that work across all current iOS devices.

Lecture

IOS300 - Auto Layout and Constraints

Objectives

  1. Create adaptive UIs using the iOS Designer
  2. Create and update constraints programmatically
  3. Animate constraint changes
  4. Use Size Classes to customize your UI for different screen sizes

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Use Auto Layout in the Designer
    Exercise
    In this exercise we will use Auto Layout to arrange several controls so they automatically size and position themselves relative to their parent and each other.
  2. Update Constraints programmatically
    Exercise
    In this exercise we will work with constraints programmatically.
  3. Animate Constraints
    Exercise
    In this exercise we'll apply UIView animations to animate constraint changes.
  4. Use Size Classes to Update UI for different screen sizes
    Exercise
    In this exercise you'll use Size Classes to make UI adjustments for larger screens.
